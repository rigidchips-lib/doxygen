Detailed Description

There are times(for example, reading from a file) when its useful to be able to tune a basic mpeg2 transport stream that contains minimal tables. this requires the least common denominator mpeg2 xport stream id information the following tune request supplies these properties. it can typiclly only be obtained from an mpeg2tunerequestfactory object. no known tune request(at this time) will return this from their createtunerequest() method. in other words, this is not the "best" tune request for any of the current tuning spaces.
